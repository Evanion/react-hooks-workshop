# Ex. 12 - Bonus (Portal with hooks)

## Instructions

The objective to this exercice is to migrate a Portal class component to a hook component.

- Clone and install Material-UI in your local environment
```sh
git clone git@github.com:mui-org/material-ui.git material-ui
cd material-ui
git checkout react-europe-exercice-12
yarn
```
- Run the documentation
```sh
yarn docs:dev
```
visit http://0.0.0.0:3000/components/portal

- Runt the tests
```sh
yarn test:watch
```
