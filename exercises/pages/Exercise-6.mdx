The strategy is to isolate the movie list and fetching the data in a component so that it can be included in a Catch.

- Create a "render props" component in MovieSearch in `src/containers/MovieDb.js`
- Integrate this component in `App` and add a` Catch` around

### Render props: implementation

export function Counter({ initialValue, children }) {
  const count = useCounter(initialValue)
  return children(count)
}
