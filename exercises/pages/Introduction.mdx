# React Advanced Workshop

## Ressources

- [Exercises on GitHub](https://github.com/smooth-code/react-hooks-workshop)
- [Slides](https://slides.react-hooks.smooth-code.com/)

## How to start the exercises?

Open a terminal, go to the chosen folder and type the following commands:

```
npm install --global smooth-code-cli
smooth-code init react-hooks-workshop
cd react-hooks-workshop
```

## Browse in the exercises

The `smooth-code-cli` command line tool that you have just installed allows you to browse in the exercises, as well as to display the answers.

```
Usage: smooth-code [command]

Commands:

  start <exercise>      Go to the beginning of an exercise
  end <exercise>        Go to the end of an exercise
  solution <exercise>   Get the solution of an exercise
```
