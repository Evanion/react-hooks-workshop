# Ex. 1 - Create React App

## Instructions

We will create a movie search application based on the [TheMovieDB API](https://developers.themoviedb.org/3).

The first step is the creation of our application using the [create-react-app](https://github.com/facebook/create-react-app/).

- Run the `npx create-react-app .` command
- Run the `npm start` command
