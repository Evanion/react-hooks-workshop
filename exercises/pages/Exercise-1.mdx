# Ex. 1 - Create React App

## Instructions

Nous allons créer une application de recherche de films basée sur l'[API de TheMovieDB](https://developers.themoviedb.org/3).

La première étape va être la création de notre application à l'aide de l'utilitaire [create-react-app](https://github.com/facebook/create-react-app/).

- Lancer la commande `npx create-react-app .`
- Lancer la commande `npm start`
