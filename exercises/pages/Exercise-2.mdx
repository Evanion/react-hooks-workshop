# Ex. 2 - Search Field

## Instructions

At first, we will focus on the search field, a simple controlled input, we will display the search typed by the user.

We will use [Smooth UI](https://smooth-ui.smooth-code.com/docs-components-input). This will allow us to have a nice site to look at without having to write a lot of CSS.

We will also use [styled-components](https://www.styled-components.com/).

- Install `styled-components @smooth-ui/core-sc`
- Create an input `controlled` component in `src/components/SearchInput.js`
- Place the `SearchInput` component in `App` and display its value using `useState`
- Delete unused files: `App.css`, `logo.svg` and `App.test.js`

```bash

npm i styled-components @smooth-ui/core-sc
```

### SearchInput.js

```js
// SearchInput.js
import React from 'react'
import { Input } from '@smooth-ui/core-sc'

export default function SearchInput(props) {
  return (
    <Input
      placeholder="Type something..."
      width="100%"
      type="search"
      {...props}
    />
  )
}
```

### How to collect the value from an input?

```js
<input
  onChange={event => {
    // An event is provided as first argument, "target" is the element that has emitted the event, it's the DOM "input" in our case.
    // An "input" has a "value" property
    console.log(event.target.value)
  }}
/>
```
